Project Overview
The core goal of this platform is to solve the problem of student procrastination and unorganized study habits. Beyond being a standard learning portal where users can access educational content, this application features a Smart Study Planner that takes input from the user (like exam dates, total topics, and daily study hours) and automatically creates a structured, day-by-day study roadmap.

🛠️ Tech Stack Used
Frontend: React.js, Vite, Tailwind CSS, Context API / Redux Toolkit

Backend: Node.js, Express.js

Database: MongoDB (Mongoose for schema modeling)

Authentication: JSON Web Tokens (JWT) & bcrypt for secure password hashing

Version Control: Git & GitHub

✨ Key Features
Smart Study Planner Algorithm: Automatically calculates and breaks down subjects into daily milestones based on user deadlines and available study hours.

Course & Content Management: Structured modules where users can view topics, check off completed lessons, and track overall progress percentages.

Interactive Dashboard: A visual representation of upcoming study tasks, completed milestones, and daily goals.

User Authentication: Secure signup/login with role-based access control (e.g., Student dashboard).

Dynamic Progress Tracking: Updates the database in real-time as users mark tasks as "Done," recalculating remaining efforts.
